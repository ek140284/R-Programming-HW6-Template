# R-Programming-HW6-Template

今回の課題は、**繰り返し処理（ループ）**の練習である。単純な繰り返しだけでなく、初学者がつまずきやすい「落とし穴」を自分で踏んで、乗り越えてもらう。

課題はすべて、`HW6.R` に記載してください。今回の課題に関係ないものが記載されても、採点されません。外部データは使用しない（ダウンロード不要）。

採点は各問2点、**満点10点**である。**値が正しく、かつそのobjectが指定されたループ（Q2 は `while`、それ以外は `for`）の中で計算されている場合のみ**2点で、どちらか一方だけでは0点である。採点プログラムはコードの構造を解析して判定するため、`sum()`・`cumsum()` などで一発計算した場合、値が合っていても0点となる。

## 課題

1. `for` ループを使って、`1` から `10` までの2乗（`1, 4, 9, ..., 100`）を計算し、長さ10の数値ベクトル `squares` として保存しなさい。

2. 元本100万円を年利4%で運用するとき、**180万円を超えるまで何年かかるか**を `while` ループで計算し、その年数を `year` として保存しなさい。

3. `for` ループを使って、`1` から `20` までの**累積和**（1, 1+2, 1+2+3, …）を計算し、長さ20の数値ベクトル `cum` として保存しなさい。`cum[20]` は `210` になるはずである。

4. 次の10年分の売上データがある（このままコピーして使うこと）。

   ```r
   sales <- c(120, 135, 128, 150, 163, 158, 171, 185, 190, 204)
   ```

   `for` ループを使って、各年の**前年比成長率（%）**を計算し、長さ10の数値ベクトル `growth` として保存しなさい。成長率は次の式で計算する。

   ```
   growth[i] = (sales[i] - sales[i - 1]) / sales[i - 1] * 100
   ```

5. 次の10人分のテスト得点には、欠損値 `NA` が含まれている（このままコピーして使うこと）。

   ```r
   scores <- c(62, 71, NA, 55, 80, NA, 90, 45, NA, 58)
   ```

   `for` ループで `scores` を1つずつ確認し、次の2つを数えなさい。

   - `n_na` … 欠損値 `NA` の個数
   - `n_over` … 60点以上の人数（`NA` は数えない）

## 提出時に必ず存在している object

採点のため、以下の object 名をすべて**完全に一致**させること。名前が1文字でも違うと、その項目は採点されない。

```r
squares   # 長さ10の数値ベクトル（1〜10の2乗）
year      # 180万円を超えるまでの年数（数値）
cum       # 長さ20の数値ベクトル（1〜20の累積和）
growth    # 長さ10の数値ベクトル（前年比成長率、1年目はNA）
n_na      # NAの個数（数値）
n_over    # 60点以上の人数（数値）
```

注意：`setwd()` を残したまま提出すると、採点サーバーではパスが異なるためエラーになり、0点になります。提出前に必ず削除するか、`#` でコメントアウトしてください。
